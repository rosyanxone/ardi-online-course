# Clone Tutorial
- Clone Repository
```bash
  git clone https://github.com/rosyanxone/ardi-online-course/
```
- Install dependencies
```bash
  composer update
```
- Start your SQL server
- Set Database name at .ENV file
```bash
    DB_DATABASE=online_course_ardi
```
- Run application
```bash
  php artisan serve
```
- Open [localhost:8000](http://127.0.0.1:8000/)
